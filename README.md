# OAC Socio-Demographic Dashboard Analysis (2021)

A Tableau-based data visualisation project analysing socio-demographic characteristics across the 2021 Output Area Classification (OAC) Supergroups.  
The dashboard integrates multiple demographic indicators to reveal structural differences between communities using coordinated visual analytics.

---

## Project Overview

Understanding demographic patterns is important for policy planning, urban development, and social research.  
This project presents a multi-view dashboard that explores socio-demographic variations across OAC Supergroups using four key indicators:

- Age distribution
- Educational attainment
- Ethnic diversity
- Household composition

The dashboard allows users to examine patterns in demographic structure and relationships between variables, supporting exploratory analysis and evidence-based decision-making.

---

## Objectives

The main objectives of this project are:

- Analyse socio-demographic characteristics across OAC Supergroups
- Compare educational attainment levels across communities
- Examine relationships between ethnic diversity and household composition
- Visualise age structure variations between groups
- Provide an interpretable dashboard for policy-oriented analysis

---

## Dataset

The analysis is based on **2021 Census Output Area Classification (OAC) data**.

The dataset includes demographic variables such as:

- Age distribution
- Education qualification levels
- Ethnic diversity indicators
- Household structure
- Dependent children statistics

These indicators allow comparison of community structures across different supergroups.

---

## Dashboard Design

The visualisation uses **four coordinated charts arranged in a structured layout** to support clear interpretation:

### 1. Grouped Bar Chart
Shows **qualification attainment levels (Level 3–4 vs Level 1–2)** across OAC Supergroups.

### 2. 100% Stacked Bar Chart
Displays **age structure composition** for each supergroup.

### 3. Scatter Plot
Explores the relationship between:

- Ethnic diversity (% multi-ethnic households)
- Dependent children in households

Reference lines highlight district averages.

### 4. Box Plot
Shows the **distribution of Level 3–4 educational attainment** across Output Areas within each supergroup.

These visualisations provide insights into both **central trends and internal variability** within demographic groups.

## Tools & Technologies

This project was developed using:

- **Tableau**
- **Microsoft Excel / CSV datasets**
- **Census 2021 OAC Data**

The dashboard design follows recognised data visualisation principles from:

- Cleveland & McGill (1984)
- Tufte (2001)
- Few (2012)
